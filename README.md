## speech-to-text

```
pip3 install SpeechRecognition
```

${HOME}/.pydistutils.cfgに以下を記述
```
[build_ext]
include_dirs=/opt/homebrew/include
library_dirs=/opt/homebrew/lib
```

```
pip3 install pyaudio
```
